快速搭建ssm项目

JAVA
发布日期:   2022-04-18

1.创建maven项目

创建完成后右键项目名称 , Add Framework Support - web , 升级为web项目

2.引入jar包

在pom.xml文件中设置打包方式为war

<packaging>war</packaging>

在<dependencies></dependencies>中引入jar包 , 所需jar包有:

  • spring
  • spring-web
  • spring-webmvc
  • spring-jdbc
  • spring-context
  • spring-core
  • spring-beans
  • spring-aop
  • spring-aspects
  • spring-expression
  • spring-massaging
  • spring-test
  • spring-tx
  • mysql-connector-java
  • standard
  • mybatis
  • mybatis-spring
  • jstl
  • javax.servlet-api
  • Junit
  • druid
  • fastjson

3.创建配置文件

在resource文件夹中创建配置文件

3.1 创建applicationContext.xml

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">

</beans>

3.2 创建database.properties

mysql 8.0以下driver=com.mysql.jdbc.driver

mysql 8.0以上driver=com.mysql.cj.jdbc.driver , url多加一项时区serverTimezone=UTC

driver=com.mysql.cj.jdbc.Driver
url=jdbc:mysql://localhost:3306/course?useSSL=true&useUnicode=true&characterEncoding=utf8&serverTimezone=UTC
user=root
pwd=123456

4.项目结构

在java文件夹中搭建项目结构

4.1 创建entity包

把数据库表转成实体类 , 属性私有化 , 生成get&set方法

4.2 创建dao包

有多少个实体类就建多少个Mapper接口

    //新增一名用户
    int addUser(User user);
    //删除一名用户
    int deleteUserById(int id);
    //更新一名用户
    int updateUser(User user);
    //查询一名用户
    User queryUserById(int id);
    //查询全部用户
    List<User> queryAllUser();

然后创建与Mapper接口同名的xml文件

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE mapper
        PUBLIC "-//mybatis.org//DTD config 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.course.dao.UserMapper">
    <insert id="addUser" parameterType="User">
        insert into course.user (uname,name,pwd,identity)
        values (#{uname},#{name},#{pwd},#{identity});
    </insert>

    <delete id="deleteUserById" parameterType="int">
        delete from course.user
        where id = #{id};
    </delete>

    <update id="updateUser" parameterType="User">
        update course.user
        set uname=#{tname},name=#{name},pwd=#{pwd},identity=#{identity}
        where id=#{id};
    </update>

    <select id="queryUserById" resultType="User">
        select * from course.user
        where id = #{id};
    </select>

    <select id="queryAllUser" resultType="User">
        select * from course.user;
    </select>
</mapper>

4.3 创建service包

有多少个mapper接口就建多少个service接口

直接将mapper接口中的代码复制到service接口里

有多少个service接口就建多少个Impl实现类

在实现类中调dao层

    private UserMapper userMapper;

    public void setUserMapper(UserMapper userMapper) {
        this.userMapper = userMapper;
    }

实现方法


    @Override
    public int addUser(User user) {
        return userMapper.addUser(user);
    }

    @Override
    public int deleteUserById(int id) {
        return userMapper.deleteUserById(id);
    }

    @Override
    public int updateUser(User user) {
        return userMapper.updateUser(user);
    }

    @Override
    public User queryUserById(int id) {
        return userMapper.queryUserById(id);
    }

    @Override
    public List<User> queryAllUser() {
        return userMapper.queryAllUser();
    }

4.4 创建controller包

controller层调用service层 , 稍后细说

5.创建mybatis-config.xml

在resource文件夹中创建mybatis-config.xml文件 , 将配置数据源交给spring去做 , 在里面配置Mapper文件即可 , dao层中有几个Mapper文件就写几个

<?xml version="1.0" encoding="UTF-8" ?>
<!DOCTYPE configuration
        PUBLIC "-//mybatis.org//DTD config 3.0//EN"
        "http://mybatis.org/dtd/mybatis-3-config.dtd">
<configuration>

    <typeAliases>
        <package name="com.course.entity"/>
    </typeAliases>

    <mappers>
        <mapper class="com.course.dao.UserMapper"/>
    </mappers>
</configuration>

6.创建spring-dao.xml文件

在resource文件夹中创建spring-dao.xml文件,主要实现以下操作

  1. 关联数据库配置文件
  2. 数据库连接池
  3. sqlSessionFactory
  4. 配置dao接口扫描包 动态实现dao接口可以注入到spring容器中
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xmlns:context="http://www.springframework.org/schema/context"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd http://www.springframework.org/schema/context https://www.springframework.org/schema/context/spring-context.xsd">
    <!-- 1.关联数据库配置文件 -->
    <context:property-placeholder location="classpath:database.properties"/>
    <!-- 2.连接池 -->
    <bean id="dataSource" class="com.alibaba.druid.pool.DruidDataSource">
        <property name="driverClassName" value="${driver}"></property>
        <property name="url" value="${url}"></property>
        <property name="username" value="${user}"></property>
        <property name="password" value="${pwd}"></property>
    </bean>
    <!-- 3.sqlSessionFactory -->
    <bean id="sqlSessionFactory" class="org.mybatis.spring.SqlSessionFactoryBean">
        <property name="dataSource" ref="dataSource"/>
        <!-- 绑定mybatis的配置文件 -->
        <property name="configLocation" value="classpath:mybatis-config.xml"></property>
    </bean>
    <!-- 4.配置dao接口扫描包,动态的实现了Dao接口可以注入到spring容器中 -->
    <bean class="org.mybatis.spring.mapper.MapperScannerConfigurer">
        <!--注入sqlSessionFactory-->
        <property name="sqlSessionFactoryBeanName" value="sqlSessionFactory"></property>
        <!--要扫描的dao包-->
        <property name="basePackage" value="com.course.dao"></property>
    </bean>
</beans>

7.创建spring-service.xml文件

在resource文件夹中创建spring-service.xml文件 , 主要实现以下操作

  1. 扫描service下的包
  2. 将所有业务类注入到spring , 可以通过xml配置 , 也可以用@Service注解实现
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xmlns:context="http://www.springframework.org/schema/context"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">
    <!--1.扫描service下的包-->
    <context:component-scan base-package="com.course.service"/>
    <!--2.将我们的所有业务类,注入到spring,可以通过配置,或者注解实现-->
    <bean id="UserServiceImpl" class="com.course.service.UserServiceImpl">
        <property name="userMapper" ref="userMapper"/>
    </bean>
</beans>

8.创建spring-mvc.xml文件

在resource文件中创建spring-mvc.xml文件 , 主要实现以下操作

  1. 注解驱动
  2. 静态资源过滤
  3. 扫描controller包
  4. 视图解析器
<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xmlns:mvc="http://www.springframework.org/schema/mvc"
       xmlns:context="http://www.springframework.org/schema/context"
       xsi:schemaLocation="http://www.springframework.org/schema/mvc
       http://www.springframework.org/schema/mvc/spring-mvc.xsd
       http://www.springframework.org/schema/context
       https://www.springframework.org/schema/context/spring-context.xsd">
    <!--1.注解驱动-->
    <mvc:annotation-driven/>
    <!--2.静态资源过滤-->
    <mvc:default-servlet-handler/>
    <!--3.扫描包:controller-->
    <context:component-scan base-package="com.course.controller"/>
    <!--4.视图解析器-->
    <bean class="org.springframework.web.servlet.view.InternalResourceViewResolver">
        <property name="prefix" value="/web-inf"/>
        <property name="suffix" value=".jsp"/>
    </bean>
</beans>

9.配置web.xml文件

配置web - WEB-INF中的web.xml文件 , 主要实现以下操作

  1. DispatchServlet
  2. 乱码过滤
  3. Session
<?xml version="1.0" encoding="UTF-8"?>
<web-app xmlns="http://xmlns.jcp.org/xml/ns/javaee"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://xmlns.jcp.org/xml/ns/javaee http://xmlns.jcp.org/xml/ns/javaee/web-app_4_0.xsd"
         version="4.0">
    <!--DispatchServlet-->
    <servlet>
        <servlet-name>springmvc</servlet-name>
        <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
        <init-param>
            <param-name>contextConfigLocation</param-name>
            <param-value>classpath:spring-mvc.xml</param-value>
        </init-param>
        <load-on-startup>1</load-on-startup>
    </servlet>
    <servlet-mapping>
        <servlet-name>springmvc</servlet-name>
        <url-pattern>/</url-pattern>
    </servlet-mapping>
    <!--乱码过滤-->
    <filter>
        <filter-name>encodingFilter</filter-name>
        <filter-class>org.springframework.web.filter.CharacterEncodingFilter</filter-class>
        <init-param>
            <param-name>encoding</param-name>
            <param-value>utf-8</param-value>
        </init-param>
    </filter>
    <filter-mapping>
        <filter-name>encodingFilter</filter-name>
        <url-pattern>/*</url-pattern>
    </filter-mapping>
    <!--Session-->
    <session-config>
        <session-timeout>15</session-timeout>
    </session-config>
</web-app>

10.配置applicationContext.xml文件

<?xml version="1.0" encoding="UTF-8"?>
<beans xmlns="http://www.springframework.org/schema/beans"
       xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
       xsi:schemaLocation="http://www.springframework.org/schema/beans http://www.springframework.org/schema/beans/spring-beans.xsd">
    
    <import resource="classpath:spring-dao.xml"/>
    <import resource="classpath:spring-service.xml"/>
    <import resource="classpath:spring-mvc.xml"/>
</beans>

11.创建jsp页面

在web - WEB-INF文件夹中创建jsp文件夹

在controller层中创建controller类 , 使用@Controller注解 , @RequestMapping(“\user”)

在controller类中调service层

@Autowired
@Qualifier("userServiceImpl")
private UserService userService;
//查询全部用户 , 并且返回一个用户展示页面
@RequestMapping("/allUser")
public String list(Model model){
  List<User> list = userService.queryAllUser();
  model.addAttribute("list",list);
  return "allUser";
}

在jsp文件夹中创建allUser.jsp页面

至此 , 一个ssm项目底层搭建完毕
